Displaying Data

Since many JavaScript frameworks also use "curly" braces to indicate a given expression should be displayed in the browser, you may use the @ symbol to inform the Blade rendering engine an expression should remain untouched. For example: 

<h1>Laravel<h1>
Hello, @{{ name }}.

In this example, the @ symbol will be removed by Blade; however, {{ name }} expression will remain untouched by the Blade engine, allowing it to be rendered by your JavaScript framework.

The @ symbol may also be used to escape Blade directives: 

{{-- Blade template --}}
@@if()
{{-- HTML Output --}}
@if()

If Statements

If Statements

Percabangan ini identik dengan percabangan yang dimiliki PHP yaitu @if, @elseif, @else, @endif contoh implementasinya seperti pada gambar berikut 

@if (count($records) === 1)
   I have one record!
@elseif (count($records) > 1)
   I have multiple records!
@else
   I don't have any records!
@endif

Untuk fleksibilitas blade menyediakan syntax baru di percabangan pada blade yaitu @unless yang merupakan kebalikan dari syntax dari percabangan ifelse implementasinya seperti pada gambar berikut 

@unless (Auth::check())
    You are not signed in.
@endunless

Selain pembahsan yang terdapat diatas, untuk kenyamanan pengguna blade juga menyediakan syntax @isset untuk memberi nilai dan syntax @empty untuk mengosongkan nilai sebuah variabel untuk implementasinya seperti gambar berikut 

@isset($records)
    // $records is defined and is not null...
@endisset

@empty($records)
    // $records is defined and is not null...
@endisset

The Loop Variable

The Loop Variable

DD

Saat melakukan iterasi melalui sebuah foreach loop, sebuah $loop variabel akan tersedia di dalam loop Anda. Variabel ini menyediakan akses ke beberapa bit informasi yang berguna seperti indeks loop saat ini dan apakah ini adalah iterasi pertama atau terakhir melalui loop 

 @foreach ($users as $user)
 @if ($loop->first)
    This is the first iteration.
 @endif
 @if ($loop->last)
    This is the last iteration.
 @endif
    This is user {{ $user->id }}
 @endforeach

Jika Anda berada dalam loop bersarang, Anda dapat mengakses $loop variabel loop induk melalui parent properti 

 @foreach ($users as $user)
   @foreach ($user->posts as $post)
     @if ($loop->parent->first)
      This is the first iteration of the parent loop. 
   @endif
  @endforeach
 @endforeach
      This is user {{ $user->id }}
 @endforeach

Beberapa variabel loop lainnya yaitu : 

$loop->index     : Indeks dari iterasi loop saat ini (dimulai dari 0).  
$loop->iteration : Iterasi loop saat ini (dimulai dari 1). 
$loop->remaining : Iterasi yang tersisa dalam loop.  
$loop->count     : Jumlah total item dalam array yang diulang.  
$loop->first     : Apakah ini adalah iterasi pertama melalui loop.  
$loop->last      : Apakah ini adalah iterasi terakhir melalui loop.  
$loop->even      : Apakah ini merupakan iterasi genap melalui loop.  
$loop->odd       : Apakah ini iterasi ganjil melalui loop.  
$loop->depth     : Tingkat bersarang dari loop saat ini.  
$loop->parent    : Saat berada dalam loop bersarang, variabel loop induk.

Conditional Classes

Conditional Classes

Direktif @class mengkompilasi string kelas CSS secara kondisional. Arahan menerima array kelas di mana kunci array berisi kelas atau kelas yang ingin Anda tambahkan, sedangkan nilainya adalah ekspresi boolean. Jika elemen array memiliki kunci numerik, elemen tersebut akan selalu disertakan dalam daftar kelas yang dirender 

@php
    $isActive = false;
    $hasError = true;
@endphp

<"span @class([
   'p-4',
   'font-bold' => $isActive,
   'text-gray-500' => ! $isActive,
   'bg-red' => $hasError, 
 ])>/span>

<"span class="p-4 text-gray-500 bg-red">

Additional Attributes

Additional Attributes

Untuk kenyamanan, Anda dapat menggunakan @checkeddirektif untuk dengan mudah menunjukkan jika input kotak centang HTML yang diberikan "dicentang". Arahan ini akan bergema checkedjika kondisi yang diberikan mengevaluasi ke true 

<'input
    type="checkbox"
    name="active"
    value="active"
   @checked(old('active', $user->active)) />

Demikian juga, @selected arahan dapat digunakan untuk menunjukkan apakah opsi pilih yang diberikan harus "dipilih": 

<'select name="version"
   @foreach ($product->versions as $version)
     <'option value="{{ $version }}" @selected(old('version') == $version)>
        {{ $version }}
    <'/option>
   @endforeach
<'/select>

Selain itu, @disabled arahan dapat digunakan untuk menunjukkan apakah elemen yang diberikan harus "dinonaktifkan": 

<' button type="submit" @disabled($errors->isNotEmpty())>Submit<'/button>

Selain itu, @readonly arahan dapat digunakan untuk menunjukkan apakah elemen yang diberikan harus "hanya-baca": 

<'input
    type="email"
    name="email"
    value="email@laravel.com"
    @readonly($user->isNotAdmin()) />

Selain itu, @required arahan dapat digunakan untuk menunjukkan apakah elemen yang diberikan harus "diperlukan" 

<'input
    type="text"
    name="title"
    value="title"
    @required($user->isAdmin()) />

The @once Directive

The @once Directive

Arahan @once memungkinkan Anda untuk menentukan bagian dari template yang hanya akan dievaluasi sekali per siklus rendering. Ini mungkin berguna untuk mendorong bagian tertentu dari JavaScript ke dalam header halaman menggunakan stacks . Misalnya, jika Anda merender komponen tertentu dalam satu lingkaran, Anda mungkin hanya ingin mendorong JavaScript ke header saat pertama kali komponen dirender: 

@once
   @push('scripts')
    <'script>
     (custom javascript)
    <'/script>
   @endpush
@endonce

Karena @once direktif sering digunakan bersama dengan @pushor @prepend direktif, @pushOnce dan @prependOnce direktif tersedia untuk kenyamanan Anda: 

@pushOnce('scripts')
    <'script>
     (custom javascript)
    <'/script>
@endPushOnce

Passing Data to Components

Passing Data to Components

Anda dapat meneruskan data ke komponen Blade menggunakan atribut HTML. Hard-coded, nilai primitif dapat diteruskan ke komponen menggunakan string atribut HTML sederhana. Ekspresi dan variabel PHP harus diteruskan ke komponen melalui atribut yang menggunakan : karakter sebagai awalan: 

<'x-alert type="error" :message="$message"/>

Anda harus mendefinisikan semua atribut data komponen dalam konstruktor kelasnya. Semua properti publik pada komponen akan secara otomatis tersedia untuk tampilan komponen. Tidak perlu meneruskan data ke tampilan dari metode komponen render: 

<'?php
namespace App\View\Components;
use Illuminate\View\Component;
   class Alert extends Component
    {
       /**
       * The alert type.
       @var string
       */
       public $message;
       /**
       * Create the component instance.
       *
       @param  string  $type
       @param  string  $message
       @return void
       */
         public function __construct($type, $message)
           {
             $this->type = $type;
             $this->message = $message;
           }
             /**
             * Get the view / contents that represent the component.
             *
             @return \Illuminate\View\View|\Closure|string
             */
         public function render()
          {
           return view('components.alert');
           }
       }

Saat komponen Anda dirender, Anda dapat menampilkan konten variabel publik komponen Anda dengan menggemakan variabel berdasarkan nama 

<'div class="alert alert-{{ $type }}">
   {{ $message }}
<'/div>

Casing

Argumen konstruktor komponen harus ditentukan menggunakan camelCase, sementara kebab-caseharus digunakan saat mereferensikan nama argumen dalam atribut HTML Anda. Misalnya, diberikan konstruktor komponen berikut: 

/**
* Create the component instance.
* @param  string  $alertType
* @return void
*/
 public function __construct($alertType)
   {
     $this->alertType = $alertType;
   }

Argumen $alertTypedapat diberikan ke komponen seperti: 

<'x-alert alert-type="danger" />

Short Attribute Syntax

Saat meneruskan atribut ke komponen, Anda juga dapat menggunakan sintaks "atribut pendek". Ini seringkali nyaman karena nama atribut sering cocok dengan nama variabel yang sesuai dengannya: 

{{-- Short attribute syntax... --}}
<'x-profile :$userId :$name />

{{-- Short attribute syntax... --}}
<'x-profile :user-id="$userId" :name="$name" />

Escaping Attribute Rendering

Karena beberapa kerangka kerja JavaScript seperti Alpine.js juga menggunakan atribut awalan titik dua, Anda dapat menggunakan :: awalan titik dua ( ) untuk memberi tahu Blade bahwa atribut tersebut bukan ekspresi PHP. Misalnya, diberikan komponen berikut: 

<'x-button ::class="{ danger: isDeleting }">
   Submit
<'/x-button>

HTML berikut akan dirender oleh Blade: 

<'button :class="{ danger: isDeleting }">
   Submit
<'/button>

Component Methods

Selain variabel publik yang tersedia untuk template komponen Anda, metode publik apa pun pada komponen dapat dipanggil. Misalnya, bayangkan sebuah komponen yang memiliki isSelectedmetode: 

/**
* Determine if the given option is the currently selected option.

*
* @param  string  $option
* @return bool
*/

 public function isSelected($option)
   {
     return $option === $this->selected;
  }

Anda dapat menjalankan metode ini dari templat komponen Anda dengan memanggil variabel yang cocok dengan nama metode: 

<'option {{ $isSelected($value) ? 'selected' : '' }} value="{{ $value }}">
   {{ $label }}
<'/option>

Hiding Attributes / Methods

Jika Anda ingin mencegah beberapa metode atau properti publik diekspos sebagai variabel ke templat komponen Anda, Anda dapat menambahkannya ke $exceptproperti array di komponen Anda 

<'?php
namespace App\View\Components;
use Illuminate\View\Component;

   class Alert extends Component
     {
       /**
         * The alert type.
         * @var string
         */ 
        public $type;
      /**
      * The properties / methods that should not be exposed to the component template.
      * @var array
      */
    protected $except = ['type'];
  }
