Slide 50

Operator “sama dengan” atau”=” merupakan salah satu operator dari Operator Perbandingan. Seperti namanya, tugas operator ini untuk membandingkan. Operator ini juga dikenal dengan operator relasi. Nilai yang dihasilkan dari operator ini berupa boolean, yaitu: true dan false. Operator yang sama dengan (==) mengembalikan true jika kedua operan memiliki nilai yang sama; jika tidak, operator akan mengembalikan false. Seperti pada contoh syntax di atas, operator “sama dengan” digunakan untuk seleksi kondisi. Yang apabila nilai variabel $a sama dengan nilai variabel $b, maka program akan mencetak kalimat “a and b are equal”.
