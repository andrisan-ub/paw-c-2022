Database

1. Akses Database Menggunakan PDO

PDO merupakan singkatan dari PHP Data Objects yang bertujuan memberikan metode seragam dalam penanganan ke beberapa database yang didukung oleh Driver PDO seperti : MySQL, PostgreSQL, Informix, MSSQL, Oracle, SQLite, Firebird/Interbase dsb. Maksud dari kata seragam mengacu kepada pembuatan script PDO yang sama untuk segala jenis database yang disupport, hanya skrip koneksi stringnya saja yang berbeda.


PDO adalah fungsi/fitur PHP yang ditulis menggunakan objek, maka untuk membuat koneksi antara PHP dengan MySQL, kita menggunakan PDO constructor, yakni dengan membuat objek baru dari class PDO (menggunakan keyword: new). Berikut adalah struktur dasar cara penulisan PDO constructor untuk database MySQL: 


                         <?php 
                            $db = new PDO($dsn, $username, $password);
                            $host = 'localhost';
                            $db = 'library';
                            $db = new PDO(“mysql:host=$host;dbname=$db”,
                              “admin”, “passwd”);

2. Cara Menampilkan Pesan Kesalahan (error) pada PDO

Karena PDO menggunakan object, penanganan kesalahan akan lebih baik jika menggunakan blok TRY…CATCH, atau dikenal dengan “exceptions”.


Dengan menggunakan penanganan kesalahan (exceptions), berikut adalah modifikasi kode PHP untuk koneksi PDO dengan MySQL: 


                         <?php 
                            $host = 'localhost';
                            
                            try {
                            $db = new PDO(“mysql:host=$host;dbname=$db”,
                                “admin”, “passwd”);
                            } catch (PDOException $e) {“admin”, “passwd”);
                              
                              die('Error: ' . $e->getMessage()');
                              }

                         <?php

                            // Returns a PDOStatement object

4. Perulangan Untuk Mengakses Tabel Database

Kita dapat melakukan perulangan melalui PDOStatement objek secara langsung untuk mendapatkan data


Dengan menggunakan perulangan foreach kita dapat mengakses semua data pada tabel SQL yang diinginkan berikut adalah contoh query foreach untuk mengakses data MySQL: 


                         <?php 
                            $sql = “SELECT * FROM student”;
                            // Returns a PDOStatement objec
                            $query = $db->query($sql);
                            foreach ($query as $row) {
                            // Column name as array index
                            // Using associative array
                            echo $row[‘nim’], ‘ ‘, $row[‘name’];
                            }

5. Fetch

Untuk mendapatkan data dari database kita bisa menggunakan method fetch() , method fetch berfungsi untuk mengambilkan satu baris data dalam satu waktu dengan menggunakan perulangan while()


berikut adalah contoh penggunaan Fetch untuk mendapatkan data dari database: 


                         <?php 
                            $sql = “SELECT * FROM student”;
                            $query = $db->query($sql);
                            while ($row = $query->fetch()) {
                            // Using indexed array
                            echo $row[0], $row[1], $row[2];
                            // Using associative array
                            echo $row[‘id’], $row[‘nim’], $row[‘name’];
                            }

6. FETCH ALL

Syntax fetchAll mengembalikan semua baris data; jadi kita harus menggunakan loop foreach untuk menghasilkan keluaran variable row dengan array index dan array associative


method fetchAll(PDO::FETCH_NUM); mengembalikan nilai semua baris data berupa array yang bisa ditampilkan dengan index angka. Berikut adalah contoh penggunaan Fetch All untuk mendapatkan variabel row dari database: 


                         <?php 
                            $sql = “SELECT * FROM student”;
                            $query = $db->query($sql);
                            $rows = $query->fetchAll();
                            foreach ($rows as $row) {
                            // Using indexed array
                            echo $row[0], $row[1], $row[2];
                            // Using associative array
                            echo $row[‘id’], $row[‘nim’], $row[‘name’];
                            }

7. FETCH_NUM

method fetchAll(PDO::FETCH_NUM); mengembalikan nilai semua baris data berupa array yang bisa ditampilkan dengan index angka. Berikut adalah contoh penggunaan FETCH_NUM untuk mengembalikan semua baris: 


                     <?php 
                        $sql = “SELECT * FROM student”;
                        $query = $db->query($sql);
                        $rows = $query->fetchAll(PDO::FETCH_NUM);
                        foreach ($rows as $row) {
                        // Can use indexed array only
                        echo $row[0], $row[1], $row[2];
                        // Cannot use associative array
                        echo $row[‘id’], $row[‘nim’], $row[‘name’];
                        }

8. FETCH_ASSOC

method fetchAll(PDO::FETCH_ASSOC); mengembalikan semua hasil query menggunakan nama kolom sebagai index array Berikut adalah contoh penggunaan FETCH_ASSOC untuk mengembalikan semua kolom: 


                   <?php 
                      $sql = “SELECT * FROM student”;
                      $query = $db->query($sql);
                      $rows = $query->fetchAll(PDO::FETCH_ASSOC);
                      foreach ($rows as $row) {
                      // Cannot use indexed array
                      echo $row[0], $row[1], $row[2];
                      // Can use associative array only
                      echo $row[‘id’], $row[‘nim’], $row[‘name’];
                      }

9. FETCH_BOTH

method fetchAll(PDO::FETCH_BOTH); mengembalikan nilai semua baris data berupa array yang bisa ditampilkan menggunakan index angka maupun index nama kolom. Berikut merupakan contoh penggunaan syntax FETCH_BOTH: 


                 <?php 
                    $sql = “SELECT * FROM student”;
                    $query = $db->query($sql);
                    $rows = $query->fetchAll(PDO::FETCH_BOTH);
                    foreach ($rows as $row) {
                    // Cannot use indexed array
                    echo $row[0], $row[1], $row[2];
                    // Can use associative array only
                    echo $row[‘id’], $row[‘nim’], $row[‘name’];
                    }

10. FETCH_OBJ

method fetchAll(PDO::FETCH_OBJ); mengembalikan sebuah objek anonim dengan nama properti yang sesuai dengan nama kolom Berikut merupakan contoh penggunaan syntax FETCH_OBJ: 


               <?php 
                  $sql = “SELECT * FROM student”;
                  $query = $db->query($sql);
                  $rows = $query->fetchAll(PDO::FETCH_OBJ);
                  foreach ($rows as $row) {
                  // Cannnot use indexed array
                  echo $row[0], $row[1], $row[2];
                  // Nor associative array
                  // Must use object-style
                  echo $row->id, $row->nim, $row->name;
                  }

11. FETCH_

Kita dapat mengikat data ke variabel dengan column index dan method fetch(PDO::FETCH_BOUND) menetapkan nilai kolom ke variabel yang ditetapkan dengan metode bindColumn() Berikut merupakan contoh penggunaan syntax FETCH_OBJ: 


             <?php 
                $sql = “SELECT * FROM student”;
                $query = $db->query($sql);
                $query->bindColumn(1, $id);
                $query->bindColumn(2, $nim);
                $query->bindColumn(3, $name);
                while ($row = $query->fetch(PDO::FETCH_BOUND)) {
                echo $id, '<br>;
                echo $nim, '<br>;
                echo $name, '<br>;
                }

Kita juga dapat mengikat data ke variabel dengan column name dan method fetch(PDO::FETCH_BOUND) menetapkan nilai kolom ke variabel yang ditetapkan dengan metode bindColumn() 


             <?php 
                $sql = “SELECT * FROM student”;
                $query = $db->query($sql);
                $query->bindColumn('id', $id);
                $query->bindColumn('nim', $nim);
                $query->bindColumn('name', $name);
                while ($row = $query->fetch(PDO::FETCH_BOUND)) {
                echo $id, '<br>;
                echo $nim, '<br>;
                echo $name, '<br>;
                }

Kita juga dapat mengikat data ke variabel dengan menggunakan column name dan column index, serta method fetch(PDO::FETCH_BOUND) menetapkan nilai kolom ke variabel yang ditetapkan dengan metode bindColumn() 


             <?php 
                $sql = “SELECT * FROM student”;
                $query = $db->query($sql);
                $query->bindColumn('id', $id);
                $query->bindColumn(2, $nim);
                $query->bindColumn('name', $name);
                while ($row = $query->fetch(PDO::FETCH_BOUND)) {
                echo $id, '<br>;
                echo $nim, '<br>;
                echo $name, '<br>;
                }

12. Memetakan Data Ke Objek

PDO dapat memetakan hasil dalam bentuk obyek. Hal ini sangat berguna jika Anda tidak ingin menggunakan lapisan abstraksi database kustom, tapi masih ingin perilaku seperti-ORM. Mari kita bayangkan bahwa kita memiliki class Student dengan beberapa properti, yang cocok dengan nama field dari database. 


           <?php 
              class Student  { 
              // must be public
               public $id;
              public $nim;
              public $name;
              
              }

              $sql = “SELECT * FROM student”;
              $query = $db->query($sql);
              $rows = $query->fetchAll(PDO::FETCH_CLASS);
              foreach ($rows as $student) {
              echo $student->id, $student->nim, $student->name;
              }

13. FETCH_FUNC

syntax FETCH_FUNC dapat digunakan jika ingin "preprocess" data dengan meneruskannya ke suatu function Berikut merupakan contoh penggunaan syntax FETCH_FUNC: 


           <?php 
              function preprocess($id, $nim, $name) {
              return “[$id] $nim: $name”;
              }
              $sql = “SELECT * FROM student”;
              $query = $db->query($sql);
              $rows = $query->fetchAll(PDO::FETCH_FUNC, ‘preprocess’);
              foreach ($rows as $row) {
              echo $row, ‘<br>’;
              }

14. UPDATE

Untuk menjalankan perintah SQL UPDATE, gunakan method exec yang mengembalikan jumlah baris yang terpengaruh Berikut merupakan contoh penggunaan syntax UPDATE: 


           <?php 
              $update = “UPDATE student SET name = ‘Lee’“ .“ WHERE nim = ‘1234’”;
                
              $count = $db->exec($update);
              // Prints the number of updated rows
              echo $count;

15. exec($delete)

exec($delete) berfungsi untuk mengembalikan jumlah nilai yang dihapus dan dapat dicetak dengan syntax “ echo $count;” Berikut merupakan contoh penggunaan syntax exec($delete): 


           <?php 
              $delete = “DELETE FROM student“ .“ WHERE nim = ‘1234’”;
              $count = $db->exec($delete);
              // Prints the number of deleted rows
              echo $count;

16. exec($insert)

exec($insert) berfungsi untuk mengembalikan jumlah nilai yang dimasukkan dan dapat dicetak dengan syntax “ echo $count;” Berikut merupakan contoh penggunaan syntax exec($insert): 


           <?php 
               $insert = “INSERT INTO student (nim, name)” .“ VALUES (‘1236’, ‘Dennis’)”;
              $count = $db->exec($insert);
              // Prints the number of inserted rows
              echo $count;

exec($insert) juga berfungsi untuk mengembalikan jumlah nilai yang dimasukkan dan dapat dicetak jumlah baris yang dimasukkan dengan syntax “ echo $count;” serta dengan menggunakan syntax “echo $db->lastInsertId();” dapat mencetak id baris baru. 


           <?php 
               $insert = “INSERT INTO student (nim, name)” .“ VALUES (‘1236’, ‘Dennis’)”;
              $count = $db->exec($insert);
              // Prints the number of inserted rows
              echo $count;
              // Prints the id of the new row
              echo $db->lastInsertId();

18. prepare($select)

prepare($select) berfungsi untuk menampilkan data, namun dengan adanya prepare statement maka query bisa dikirim secara terpisah antara query inti dengan data dari query agar lebih cepat dan aman Berikut merupakan contoh penggunaan syntax prepare($select): 


           <?php 
              $select = “SELECT * FROM student”;
              $prepared = $db->prepare($select);
              $prepared->execute();
              $rows = $prepared->fetchAll(PDO::FETCH_OBJ);
              foreach ($rows as $row) {
              echo $row->id, '<br>';
              echo $row->nim, '<br>';
              echo $row->name, '<br>';
              }

19. prepare($insert)

prepare($insert) berfungsi untuk memasukkan data baru, namun dengan adanya prepare statement maka query bisa dikirim secara terpisah antara query inti dengan data dari query agar lebih cepat dan aman. Berikut merupakan contoh penggunaan syntax prepare($insert): 


         <?php 
            $insert = “INSERT INTO student (nim, name)” .“ VALUES (‘1236’, ‘Dennis’)”;
            $prepared = $db->prepare($insert);
            $prepared->execute();

20. prepare($update)

prepare($update) berfungsi untuk memperbarui data yang sudah ditambahkan pada tabel, namun dengan adanya prepare statement maka query bisa dikirim secara terpisah antara query inti dengan data dari query agar lebih cepat dan aman Berikut merupakan contoh penggunaan syntax prepare($update): 


       <?php 
          $delete = “DELETE FROM student WHERE nim = ‘1234’”;
          $prepared = $db->prepare($delete);
          $prepared->execute();

21. Parameter Tanda Tanya

Cara ini menggunakan teknologi prepared statement. Perintah sql diketik dalam sebuah template, kemudian nilai dari parameter di dalam perintah tersebut dikirim, dan baru pada akhirnya perintah tersebut dieksekusi. arameter tanda tanya lebih mudah digunakan, akan tetapi rentan dengan kesalahan pengurutan. Berikut merupakan contoh penggunaan parameter tanda tanya: 


         <?php 
            // Using question-mark parameter
            $insert = “INSERT INTO student (nim, name) VALUES (?, ?)”;
            $prepared = $db->prepare($insert);
            // With bindValue() method
            $prepared->bindValue(1, ‘1234’);
            $prepared->bindValue(2, ‘Smith’);
            $prepared->execute();

Pada contoh di atas, pertama-tama objek $db (PDO) menyiapkan template perintah INSERT dan disimpan dalam objek $insert (PDOStatement). Tanda tanya di dalam template kemudian diberi nilai dengan perintah bindValue. 1, 2 pada bindParam mewakili urutan parameter di dalam template. Pada akhirnya, saat $prepare->execute(), perintah SQL benar-benar dikirim dan dieksekusi. Perhatikan bahwa kita menggunakan $prepare->execute(), bukan $db->exec(). Keduanya merupakan fungsi yang berbeda pada class yang berbeda. Pada contoh di atas, $id, $tipe, $kategori dan $harga tidak perlu diproses dengan PDO::quote() penggunaan metode prepared statement otomatis telah mengamankan query dari SQL Injection.

22. Parameter Bernama

Parameter bernama lebih sulit diketik, tetapi lebih kebal terhadap kesalahan pengurutan. Berikut merupakan contoh penggunaan parameter bernama: 


       <?php 
          // Using named parameter
          $insert = “INSERT INTO student (nim, name) VALUES (:nim, :name)”;
          $prepared = $db->prepare($insert);
          // Directly to the execute() method
          $prepared->execute(
          array(
          ‘nim’ => ‘1234’,
          ‘name’ => ‘Smith’
          )
          );

23. Akses Database Menggunakan SQLI

SQLI MySQLi merupakan singkatan dari MySQL Improved. Ekstensi MySQL ini menambahkan fitur baru di interface database MySQL. MySQLi bersifat prosedural dan juga berorientasi objek, di mana siftat prosedural adalah atribut yang diwariskan dari versi MySQL yang lebih lama.


MySQL versi awal membagi task ke dalam satu prosedur bertahap (langkah demi langkah) dan saling berhubungan. Tindakan modifikasi akan semakin sulit karena kode harus diubah dari bagian paling pertama. Di sisi lain, MySQLi melihat data sebagai satu set objek yang dapat ditukarkan dan memiliki fungsi serta memperbolehkan pengguna untuk menambah atau menghapus data.

a. MySQLI dengan menggunakan procedural programming style


       <?php 
          // Procedural style
          $mysqli = mysqli_connect($host, $user, $pass, $db);
          if (mysqli_connect_errno($mysqli)) {
          echo “Error: “, mysqli_connect_error();
          }

b. MySQLI dengan menggunakan procedural programming style


       <?php 
          // Object-oriented style
          $mysqli = mysqli_connect($host, $user, $pass, $db);
          if (mysqli_connect_errno($mysqli)) {
          echo “Error: “, mysqli_connect_error();
          }
