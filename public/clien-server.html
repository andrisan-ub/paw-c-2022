Pengenalan PHP: Client-Server

PHP adalah kependekan dari Hypertext Preprocessor. Sebuah server-side script yang banyak digunakan untuk mengembangkan website dan biasa disisipkan ke dalam file HTML. PHP disebut sebagai server-side script karena proses eksekusi kode dasar programnya dilakukan pada sisi server. Sehingga, client (web browser) hanya akan menerima hasil dari PHP script yang dijalankan berupa plain HTML, namun client tidak akan mengetahui kode dasarnya.

Berikut merupakan gambaran bahasa PHP sebagai server side:

<!DOCTYPE html>
                <html lang="en">
                <head>
                  <meta charset="UTF-8">
                  <meta http-equiv="X-UA-Compatible" content="IE=edge">
                  <meta name="viewport" content="width=device-width, initial-scale=1.0">
                  <title>Client Server PHP</title> 
                </head>
                <body>
                  <h1>
                <?php
               $Hello  = "Hello"; 
               $World  = "World"; 
               echo   $Hello, $World;
              <?>
                </h1>
                </body>
                </html>
