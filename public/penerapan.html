Seleksi Kondisi, Looping, dan Array

1. Seleksi Kondisi

a. IF

Pada logika if, suatu perintah akan dijalankan hanya jika kondisi pada logika if bernilai benar. Cara penulisan if : 


                    if(kondisi){
                        //kode yang akan dijalankan jika kondisi bernilai benar;
                    }

Contoh penerapan seleksi kondisi "if" sebagai berikut:


                     <?php 
                        $a = 10;
                        $b = 10;
                        if($a = $b) {
                        echo "nilai a sama dengan nilai b";
                    }
                    echo "<br>";
                    echo "nilai a = ", $a;
                    echo "<br>";
                    echo "nilai b = ", $b;

b. IF ELSE

Prinsipnya sama dengan logika if, hanya saja ditambahkan sebuah perintah lagi jika kondisi yang dicek bernilai salah. Jadi pada logika if else disiapkan dua statement perintah untuk menangani sebuah kondisi saat bernilai benar dan saat bernilai salah. Cara penulisan if else: 


                    if(kondisi){
                        //kode yang akan dijalankan jika kondisi bernilai benar;
                    } else {
                        //kode yang akan dijalankan jika kondisi bernilai salah;
                        }

Contoh penerapan seleksi kondisi "if else" sebagai berikut:


                     <?php 
                        $a = 10;
                        $b = 20;
                        if($a < $b) {
                        echo "nilai a lebih kecil dari nilai b";
                    } else{
                        echo "nilai a lebih besar dari nilai b";
                    }
                    echo "<br>";
                    echo "nilai a = ", $a;
                    echo "<br>";
                    echo "nilai b = ", $b;

c. IF ELSE IF

if else if merupakan logika untuk menangani beberapa kondisi yang berada dalam satu rutin pengecekan. Dalam logika if else if perintah pada masing-masing if akan dijalankan hanya jika kondisi didalam logika if masing-masing bernilai benar. Pada akhir else if bisa ditambahkan else saja untuk menangani jika semua kondisi if else if di atasnya bernilai salah. Cara penulisan if else if : 


                    if(kondisi){
                        //kode yang akan dijalankan jika kondisi 1 bernilai benar;
                    } elseif {
                        //kode yang akan dijalankan jika kondisi 1 bernilai salah dan kondisi 2 bernilai benar;
                        } else{
                        //kode yang akan dijalankan jika kondisi 1 dan kondisi 2 bernilai salah;
                        }

Contoh penerapan seleksi kondisi "if else if" sebagai berikut:


                     <?php 
                        $a = 20;
                        $b = 20;
                        if($a < $b) {
                        echo "nilai a lebih kecil dari nilai b";
                    } elseif ($a > $b){
                        echo "nilai a lebih besar dari nilai b";
                        } else($a > $b){
                        echo "nilai a sama dengan nilai b";
                    }
                    echo "<br>";
                    echo "nilai a = ", $a;
                    echo "<br>";
                    echo "nilai b = ", $b;

d. Logika Khusus

Ini merupakan penulisan untuk seleksi kondisi yang lebih simpel dari IF. Suatu kondisi dicek nilai benar atau salahnya lalu jika benar mengerjakan blok perintah benar lalu jika salah mengerjakan blok perintah salah. Kelemahan dari penggunaan logika khusus adalah hanya bisa menaruh satu statement perintah pada blok benar dan blok salah. Cara penulisan logika khusus : 


                    (kondisi) ? statement jika benar : statement jika salah;

e. Switch

switch digunakan untuk menyeleksi banyak kondisi. Prinsipnya sama dengan if else if namun tujuan adanya logika switch adalah untuk membuat script lebih efisien dan juga memudahkan pembacaan script. Cara penulisan switch : 


                     <?php 
                    $angka = 3;
                    switch(angka){
                    case 1:
                    echo "isi variabel angka adalah 1";
                    break;
                    case 2:
                    echo "isi variabel angka adalah 2";
                    break;
                    case 3:
                    echo "isi variabel angka adalah 3";
                    break;
                    case 4:
                    echo "isi variabel angka adalah 4";
                    break;
                    case 5:
                    echo "isi variabel angka adalah 5";
                    break;
                    default:
                    echo "isi variabel angka tidak ditemukan";
                    break;
                    }
