PHP Dasar: Pengenalan Syntax Dasar PHP

Syntax PHP adalah aturan penulisan agar mampu dimengerti dengan benar oleh compiler saat membaca bahasa pemrograman.

1. Semmicolons

semicolons atau titik koma (;) pada php berfungsi untuk memisahkan statement yang biasa terletak pada akhir perintah dan tanda ini harus ada apabila tidak ada kemungkinan akan terjadi error. Berikut contoh penggunaan semicolons(;) 


                <?php
                echo "Hello, world!"; 
                $myFunction =(42, "O'Reilly" );
                $a = 1;
                $name = "Elphaba"; 
                $b = $a / 25.0 ;  
                if ($a == $b){
                echo "Rhyme ? And Reason";  
                }
                 ?>

2. Dollar Sign

Cara penulisan variabel dalam php yang benar adalah harus diawali dengan tanda dollar atau dollar sign kemudian diikuti dengan huruf dan beberapa juga menggunakan awalan karakter underscore. Jadi, setelah tanda $ pada penulisan variabel php maka harus diikuti dengan karakter pertama berupa huruf atau underscore (_), dan karakter berikutnya dan seterusnya dapat menggunakan huruf, angka atau underscore (_) pula. Ada aturan penting yan harus diperhatikan dalam penulisan variabel php, yaitu bahwa variabel di dalam php tidak boleh diawali dengan angka, dan memiliki minimal panjang variabel adalah 1 karakter tidak termasuk tanda $.


                <?php
                $bill;
                $head_count;
                $MaximumForce;
                $I_HEART_PHP;
                $underscore;
                $_int;
                 ?>

3. Passing by Reference

Reference sebuah nilai variabel yang sudah terisi di variabel lain. Jadi dalam PHP kita bisa mereferensikan sebuah variabel yang sudah ada nilainya ke dalam varaibel dengan nama yang sama tetapi mengeluarkan output yang beda dengan cara sebagai berikut:


                <?php
                $foo ="bar";
                $$foo ="world";
                $$$foo =100;
                 ?>

Dengan melihat kode di atas, dapatkah teman - teman simpulkan pada variabel apa yang memiliki nilai 100? dengan teori reference variable yang telah dipelajari, maka dapat ditentukan bahwa variabel yang memiliki nilai 100 adalah $world.

4. Operasi Aritmatika

Operasi aritmatika merupakan operator matematisyang terdiri dari penambahan, pengurangan, perkalian, pembagian, dan modulus. Operasi aritmatika dalam PHP tidak jauh beda dengan operasi aritmatika yang ada di Java. Agar lebih mengerti mungkin bisa iat cotoh di bawah ini:


                <?php
                $a = 3 + 4 + 5 -2;
                echo $a
                 ?>

Output yang ditampilkan pada website adalah hasil operasi dari variabel a yaitu 10.

5. Operator Perbandingan

Di PHP, operator perbandingan mengambil nilai sederhana ( angka atau string ) sebagai argumen dan mengevaluasi ke TRUE atau FALSE. Berikut adalah daftar operator perbandingan. Agar lebih mengerti mungkin bisa iat cotoh di bawah ini:


                <?php
                ($x==$y) //operator ini bernilai benar jika $x sama persis dengan $y
                ($x===$y) //operator ini bernilai benar jika $x sama persis dengan $y, dan keduanya bertipe sama
                ($x != $y) //operator ini bernilai benar jika $x tidak sama persis dengan $y.
                ($x <> $y) //operator ini bernilai benar jika $x tidak sama persis dengan $y.
                ($x !== $y) //operator ini bernilai benar jika $x tidak sama dengan $y, atau keduanya tidak bertipe sama.
                ($x < $y) //operator ini bernilai benar jika $x (argumen tangan kiri) benar-benar kurang dari $y (argumen tangan kanan).
                ($x > $y) //operator ini bernilai benar jika $x (argumen tangan kiri) benar-benar lebih besar dari $y (argumen tangan kanan).
                ($x <= $y) //operator ini bernilai benar jika $x (argumen tangan kiri) benar-benar kurang dari atau sama dengan $y (argumen tangan kanan).
                ($x >= $y) //operator ini bernilai benar jika $x (argumen tangan kiri) benar-benar lebih besar dari atau sama dengan $y (argumen tangan kanan).
                echo $a
                 ?>

5. Operator Logika

Operator Logika adalah operator yang digunakan untuk membandingkan 2 kondisi logika, yaitu logika benar (TRUE) dan logika salah (FALSE). Operator logika sering digunakan untuk kodisi IF, atau untuk keluar dari proses perulangan (looping).


                <?php
                ($a and $b) //operator ini bernilai benar jika $a dan $b bernilai true
                ($a && $b) //operator ini bernilai benar jika $a dan $b bernilai true
                ($a or $b) //operator ini bernilai benar jika salah satu dari $a atau $b bernilai true
                ($a || $b) //operator ini bernilai benar jika salah satu dari $a atau $b bernilai true
                ($a xor $b) //operator ini bernilai benar jika salah satu dari $a atau $b bernilai true, tetapi tidak keduanya
                (!$a) //operator ini bernilai benar jika $a bernilai false
                echo $a
                 ?>

6. Operator Spaceship

Spaceship operator atau disebut juga three way comparison operator adalah operator perbandingan yang memiliki 3 kemungkinan return value: 0 jika yang dibandingkan setara, -1 jika nilai sebelah kiri lebih kecil, dan 1 jika nilai sebelah kiri lebih besar.


                <?php
                var_dump(5 <=> 6); //operator ini akan menghasilkan output int (-1)
                var_dump(1 <=> 1); //operator ini akan menghasilkan output int (0)
                var_dump(5 <=> 2); //operator ini akan menghasilkan output int (1)
                 ?>

7. Interpolasi Variabel PHP

Interpolasi variabel adalah menambahkan variabel di antaranya saat menentukan literal string. PHP akan mengurai variabel interpolasi dan mengganti variabel dengan nilainya saat memproses string literal.


                <?php
                $name = 'Guido'; 
                echo “Hi, $name”; 
                echo 'Hi, $name'; 
                 ?>

Cara lainnya adalah dengan mengelilingi variabel menjadi diinterpolasi dengan kurung kurawal


                  <?php
                  $n = 12; 
                  echo “You are the $n th person”; 
                  echo "You are the {$n}th person"; 
                   ?>

8. Index String

setiap karakter String memiliki indeks


                <?php
                $s = 'Hello'; 
                echo $s[0]; //output syntax ini akan menampilkan karakter "H" 
                echo $$[-1]; //output syntax ini akan menampilkan karakter "o", index -1 akan mengambil karakter dari urutan belakang 
                 ?>

9. strlen()

Fungsi strlen() mendapatkan panjang string


                <?php
                $s = 'FILKOM UB'; 
                echo strlen($s) //output syntax ini akan menampilkan 9 yang berarti jumlah karakter dalam variabel $s adalah 9 
                 ?>

10. strtoupper()

Fungsi strtoupper() mengonversi semua karakter dalam string ke huruf besar


                <?php
                $s = 'filkom ub'; 
                echo strtoupper($s) //output syntax ini akan menampilkan FILKOM UB 
                 ?>

11. strtolower()

Syntax tersebut menggunakan kode The strtolower()yang memiliki fungsi untuk mengkonversi semua karakter nilai variabel tersebut dari uppercase menjadi lowercase, seperti contohnya pada PPT yaitu variabel ‘FILKOM UB’ pada outputnya akan berubah menjadi ‘filkom ub’. 


                <?php
                $s = 'FILKOM UB'; 
                echo strtolower($s) //output syntax ini akan menampilkan filkom ub 
                 ?>

12. trim()

Syntax The trim()memiliki fungsi untuk menghapus semua spasi kosong yang ada di sekitar isi dari suatu variabel. Contohnya pada syntax diatas yang memiliki isi dari variabel yaitu ‘ FILKOM UB ‘ ketika menggunakan kode The trim() maka spasi kosong yang ada di sebelah kiri huruf F dan sebelah kanan huruf B akan dihilangkan, sehingga isi dari variabel tersebut akan berubah menjadi ‘FILKOM UB’. 


                <?php
                $s = ' FILKOM UB '; 
                echo trim($s) //output syntax ini akan menampilkan FILKOM UB 
                 ?>

13. concatenate()

Syntax The concatenate memiliki fungsi untuk menggabungkan karakter pada variabel string. Contohnya kita bisa lihat di atas yaitu terdapat variabel $s1 yang berisikan ‘FILKOM’ kemudian terdapat variabel $s2 yang berisikan ‘UB’. Maka, dengan menggunakan kode The concatenate kita dapat menyatukan isi dari kedua variabel tersebut. Contohnya dari hasil penggunaan kode tersebut dapat dilihat pada gambar di atas dengan syntax $s3 = $s1 . $s2 yang nantinya akan menghasilkan output berupa ‘FILKOM UB’. 


                <?php
                $s1 = 'FILKOM'; 
                $s2 = 'UB'; 
                echo $s1.$s2 //output syntax ini akan menampilkan FILKOM UB yang berarti menggabungkan 2 variabel string 
                 ?>

14. is_bool()

Syntax is_bool() memiliki fungsi untuk mengecek apakah variabel tersebut merupakan tipe data boolean atau tidak. Untuk penulisannya, deklarasikan variabel $s1 terlebih dahulu, kemudian kita dapat mengecek apakah variabel tersebut merupakan tipe data boolean atau tidak dengan menggunakan echo “...” . is_bool($s1). Jika nilai variabel $s1 merupakan tipe data boolean maka outputnya akan menghasilkan nilai 1 dan jika tidak maka outputnya akan kosong atau tidak menampilkan apa-apa

15. unset()

Syntax unset() memiliki fungsi untuk menghapus sebuah variabel tertentu. Pada gambar diatas awalnya terdapat isi dari variabel $name yaitu “Fred”, kemudian ketika kita ingin menghapus isi dari sebuah variabel, kita dapat menggunakan unset() yang diikuti dengan nama variabel yang ingin dihapus. Setelah itu dijalankan dengan menggunakan echo $namavariable. Maka output dari syntax diatas adalah kosong. 


                <?php
                $name = 'Fred'; 
                unset($name); 
                echo $name //output syntax ini akan menampilkan kosong 
                 ?>

16. isset()

Syntax isset() memiliki fungsi untuk mengeset suatu variabel. Selain itu kode isset() juga berfungsi untuk mengecek apakah nilai dari variabel tersebut kosong (null) atau tidak.. Contohnya pada gambar diatas terdapat isi dari variabel $s1 dengan menggunakan isset() maka hasil nya akan salah karena sebelumnya variabel $name belum diset atau dideklarasikan. Kemudian setelah variabel $name dideklarasikan menjadi “Fred” maka setelah dicek menggunakan isset() maka hasilnya akan benar mengingat variabel $name sudah dideklarasikan sebelumnya. 


                <?php
                $s1 = isset($name); //syntax ini bernilai false karena $name belum dideklarasikan 
                $name = "Fred";
                $s2 = isset($name); //syntax ini bernilai true karena $name telah dideklarasikan.
                 ?>
