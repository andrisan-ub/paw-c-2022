Introduction

Blade adalah template yang disediakan oleh Laravel untuk memudahkan pemrogram dalam membuat program php. File blade memiliki extension .blade.php dan disimpan dalam directory resources/views. Blade template dapat digunakan bersamaan dengan program php dan framework javascript.

Tanda kurung kurawal {} digunakan untuk menampilkan data di dalam file blade

Dalam program php, untuk menampilkan data digunakan keyword echo. Blade template dapat mempersingkat kode php, karena cukup menggunakan tanda kurung kurawal {}


Blade Directives

Blade template memiliki keyword yang mirip dengan keyword pada php. Setiap keyword pada blade template diawali dengan tanda @

Percabangan

  • if else
    • Percabangan if else pada blade template menggunakan keyword @if, @elseif, @else, @endif.
    Blade template juga menyediakan keyword @unless. Baris kode di antara keyword @unless dan @endunless akan dijalankan jika kondisi di dalam tanda kurung tidak terpenuhi.
  • auth
    • Keyword @auth dan @endauth berfungsi untuk memeriksa apakah pengguna telah terautentifikasi. Sementara itu, keyword @guest dan @endguest digunakan untuk memeriksa apakah pengguna belum terautentifikasi.
  • enviroment
    • Kode di antara keyword @env dan @endenv akan dieksekusi jika aplikasi dijalankan dalam lingkungan yang sesuai dengan kondisi di dalam tanda kurung.
  • section
    • Keyword @hasSection dan endif digunakan untuk memeriksa apakah suatu section memiliki isi. Sementara itu, keyword @sectionMissing dan @endif digunakan untuk memeriksa apakah section tidak memiliki isi.
  • switch
    • Percabangan switch pada blade template sama seperti percabangan switch pada program php. Keyword yang digunakan pada percabangan switch terdiri dari @switch, @case, @break, @default, dan @endswitch

Perulangan

  • for
    • Pengulangan for menggunakan keyword @for dan endfor.
  • foreach
    • Pengulangan foreach menggunakan keyword @foreach dan @endforeach.
  • forelse
    • Pengulangan forelse menggunakan keyword @forelse, @empty, dan @endforelse.
  • while
    • Pengulangan while menggunakan keyword @while, dan @endwhile.
  • continue dan break
    • Keyword @continue digunakan untuk berhenti dari loop yang sedang berlangsung dan melanjutkan loop selanjutnya. Sementara itu, keyword @break digunakan untuk keluar dari loop.
