Interpolasi adalah proses menampilkan value dari variable. Interpolasi dapat dilakukan dengan meletakkan variabel pada kutip 2, atau menempatkan variabel di dalam ({}). Pada contoh program, line 1 dan line 3 varibel akan ditampilkan dengan namanya, karena variabel diletakkan pada petik 1. Sementara pada line 2 dan 4 variabel akan ditampilkan sebagai valuenya karena diletakkan dalam petik 2.


String merupakan tipe data yang serupa dengan array. Setiap character string dapat dianggap sebagai elemen array, Oleh karena itu kita dapat mengakses string menggunakan index untuk mendapatkan salah satu characternya.


Kita dapat membandingkan 2 karakter String menggunakan == operator. Perbandingan ini dilakukan untuk melihat apakah ke 2 String memiliki nilai yang sama.
