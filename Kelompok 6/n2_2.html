Studi kasus PHP basic (slide 22)


Apa itu variabel ? Variabel dalam suatu program digunakan untuk menyimpan suatu nilai (value) atau data yang dapat digunakan nanti dalam suatu program. PHP memiliki cara sendiri untuk mendeklarasikan dan menyimpan variabel. Dalam PHP ada beberapa aturan untuk membuat variabel yang benar atau dianggap legal. Berikut ini merupakan beberapa contoh variabel yang benar dan salah.


Ciri - ciri variabel:

  1. Diawali dengan tanda dollar ($).
  2. Variabel hanya boleh mengandung alphanumeric dan underscore (A-Z, 0-9, dan “_”) saja pada namanya.
  3. Nama variabel dalam PHP harus diawali dengan huruf bukan angka. Variabel di PHP bersifat Case-Sensitive.
  4. Nama variabel diletakkan sebelum tanda sama dengan (=), dan value atau nilainya diletakkan setelah tanda sama dengan (=).
  5. Tidak boleh mengandung unsur spasi.
  6. Nama variabel dalam PHP harus diawali dengan huruf bukan angka.


Contoh variable benar:

  1. $bill
  2. $head_count
  3. $MaximumForce
  4. $I_HEART_PHP
  5. $_underscore
  6. $_Int


Contoh variable salah:

  1. $not valid
  2. $|
  3. $3wa
