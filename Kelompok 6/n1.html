Pada materi ini kita belajar tentang dasar dasar penulisan PHP dan cara menampilkan output menggunakan PHP. Kode PHP bisa dituliskan di dalam file HTML maupun file PHP itu sendiri, namun setiap file yang memiliki kode PHP harus memiliki ekstensi .php. Kode PHP harus dituliskan di dalam tag php. Untuk menampilkan output pada PHP kita bisa menggunakan kode print atau echo. Pada materi ini script PHP dijalankan pada server side. PHP bisa diproses dengan 3 cara yaitu server-side scripting, command-line scripting, dan client-side application. Script PHP yang ada akan diproses oleh server dan dikembalikan dalam bentuk HTML.
