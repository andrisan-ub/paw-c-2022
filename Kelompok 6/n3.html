Seleksi Kondisi berguna untuk melakukan suatu aksi ketika kondisi tertentu terjadi. terdapat 3 jenis seleksi kondisi pada PHP:

  1. If Statement
    If statement akan mengeksekusi kode jika kondisi bernilai true:
    if (condition) {
    code to be executed if condition is true;
    }
  2. If Else Statement
    Jika kondisi true ada kode yang akan dieksekusi, jika salah maka akan ada kondisi yang dieksekusi juga:
    if (condition) {
    code to be executed if condition is true;
    } else {
    code to be executed if condition is false;
    }
  3. If Else If Statement
    Statement ini akan menyeleksi beberapa kondisi:
    if (condition) {
    code to be executed if this condition is true;
    } elseif (condition) {
    code to be executed if first condition is false and this condition is true;
    } else {
    code to be executed if all conditions are false;
    }

Looping berguna untuk mempersingkat codingan ketika kita ingin menjalankan kode yang sama berulang kali. Terdapat 4 jenis loop pada PHP:

  1. While
    Perulangan akan terus terjadi selama condition bernilai true
  2. Do While
    Sama seperti while, bedanya kode pasti akan dijalankan sekali
  3. For
    For loop biasanya digunakan ketika kita tahu ingin melakukan berapa kali perulangan
  4. foreach
    Looping akan dilakukan melalui elemen elemen array

Array berfungsi untuk menyimpan banyak value dalam 1 variabel. Terdapat 3 jenis array dalam PHP:

  1. Indexed array yaitu array dengan numerik index
  2. Associative array yaitu array dengan keys
  3. Multidimensional Array yaitu array yang berisi lebih dari 1 array

Pada contoh kami membuat sebuah array yang menampung nama nama bunga, lalu kami menggunakan looping yang mengakses setiap elemen array, lalu di setiap elemen array kami menggunakan selection condition dimana jika nama bunga pada elemen adalah melati atau anggrek atau bangkai maka akan keluar output Nama tumbuhan sudah benar, dan Jika salah maka akan keluar output nama bunga yang ada di elemen array.
