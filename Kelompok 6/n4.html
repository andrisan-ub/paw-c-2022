Penerapan materi terhadap nilai konversi mahasiswa seperti terlampir di bawah ini: (memanfaatkan if, switch, dan loop)!


Penerapan konversi yang kami buat menggunakan array, looping, dan selection condition. Kami menggunakan associative array dengan key sebagai nama mahasiswa, dan value sebagai niilai mahasiswa. Array ini berfungsi untuk menampung data dari mahasiswa. Looping yang kami gunakan adalah foreach loop, alasan dari penggunaan foreach loop adalah karena kita ingin mengambil value dari elemen array, sehingga loop menjadi lebih praktis untuk diaplikasikan. Untuk mengkonversi nilai mahasiswa kami menggunakan selection condition yaitu if else. Jika nilai mahasiswa ada di rentang tertentu, maka konversi akan dihasilkan sesuai dengan ketentuan yang ada.
