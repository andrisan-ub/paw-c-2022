Studi kasus PHP basic (slide 14)


Seperti pada bahasa pemrograman pada umumnya, terdapat beberapa ketentuan dasar. Pada baris pertama terdapat perintah untuk melakukan print dengan mengetik syntax echo. Pada baris kedua terdapat fungsi . Pada baris ketiga dan keempat terdapat dua variabel yaitu $a dengan nilai ‘1’ dan $name dengan nilai ‘Elphaba’. Pada php variabel dideklarasikan dengan menggunakan tanda “$”. Variabel pada php cukup flexible dimana kita hanya perlu memberi tanda “ ‘’ “ pada nilai untuk menyatakan tipe data string. Untuk tipe data integer, double, float, kita tidak perlu memberi tanda apapun. Sistem akan menyatakan tipe data variabel secara otomatis. Pada baris kelima terdapat juga variabel yaitu $b dimana nilainya berupa operator pembagian variabel $a terhadap 25.0. Pada baris keenam hingga kedelapan terdapat percabangan
