PHP Basics

Slide 14

PHP merupakan bahasa pemrograman yang menggunakan semicolon untuk memisahkan setiap statemen code.

Slide 22

Nama variabel PHP selalu diawali dengan dollar sign($) dan case-sensitive. Nama yang valid selalu dengan awalan huruf atau underscore ( _ ) dan dipisahkan dengan underscore ( _ ).

Slide 26

Variabel $world yang mengandung nilai 100. Hal ini dikarenakan pada PHP, kita dapat mereferensikan nilai dari sebuah variabel dimana namanya disimpan dalam variabel lain.

Slide 34-36

PHP memiliki arithmetic operator dan assignment operator yang mirip dengan bahasa pemrograman Java tetapi terdapat perbedaan yaitu arithmetic operator concat (.).

Slide 38-42

Melanjutkan dari poin sebelumnya, PHP juga memiliki comparison operator dan logical operator yang mirip di Java. PHP 7 memperkenalkan spaceship operator yang berfungsi untuk membandingkan dua ekspresi dan null coalescing operator sebagai pengganti ternary operation di konjungsi dengan isset() function. Dalam membandingkan nilai-nilai, diperlukan kehati-hatian terhadap type juggling.

Slide 47-50
  • Dalam menampilkan nilai dari variabel kita dapat menggunakan Interpolation yaitu proses mengganti nama variabel menjadi nilainya dalam suatu string.Kita juga dapat melakukan interpolasi dengan kurung kurawal ({}).
  • String itu seperti array, setiap karakter huruf memiliki indeks yang dimulai dari 0.
  • Untuk tes apakah dua String itu sama dapat menggunakan operator == .
Slide 52-56
  • Function strlen() untuk mengetahui panjang string
  • Function strtolower() untuk merubah semua huruf lowercase di string
  • Function strtoupper() untuk merubah semua huruf uppercase di string
  • Function trim() untuk menghapus whitespace yang mengelilingi string
  • Untuk menggabungkan (Concanate) string, diperlukan dot (.) operator
Slide 58-59
  • function unset() untuk menghapus variabel
  • function isset() untuk mengetahui apakah variabel memiliki nilai atau tidak
