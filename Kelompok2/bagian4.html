Program Konversi Nilai

Program ini adalah implementasi dari dari penerapan materi. Untuk melihat lebih lanjut dapat membuka Program Konversi Nilai dengan menggunakan IDE atau code editor.

Di program ini, kami memanfaatkan if untuk memilih merubah nilai menjadi angka mutu dan huruf mutu, switch-case untuk meng-output angka mutu/huruf mutu dari nilai yang telah di input, dan Do-While Looping untuk mengetahui user ingin menggunakan program kembali.
