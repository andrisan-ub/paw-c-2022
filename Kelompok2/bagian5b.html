AJAX Chat

Pada bagian ini, kami mempraktikkan source code pada materi Form Handling yakni AJAX app chat.

AJAX app chat adalah hasil aplikasi web yang interaktif dari teknik pemrograman berbasis web. AJAX app chat ini memiliki fungsi seperti aplikasi chat biasanya. Pada awalnya kita harus masuk ke dalam web tersebut, lalu memasukkan nama dan isi pesan yang akan dikirimkan. Selain itu, jika telah dikirimkan. AJAX app chat juga dapat memberikan pemberitahuan bahwa chat anda telah dibaca dengan tanda read di sekitar chat tersebut. Isi chat-chat tersebut dapat langsung tersimpan dalam chat.txt


Kurang lebih seperti inilah tampilan yang diberikan oleh web AJAX app chat.
