Laravel Blade Template

Definisi Blade

Blade adalah fitur yang disediakan Laravel untuk proses templating sederhana namun sangat bermanfaat dalam proses pengembangan tampilan halaman web. Tidak seperti fitur templating PHP populer lainnya, Blade tidak membatasi programmer untuk menggunakan kode PHP biasa dalam membuat kode untuk tampilan. File template blade menggunakan ekstensi file .blade.php dan biasanya disimpan di direktori resources/views.


Routing Blade

Tampilan blade dapat dikembalikan dari rute atau pengontrol menggunakan bantuan tampilan global. Tentu saja, seperti yang disebutkan dalam dokumentasi tentang tampilan, data dapat diteruskan ke tampilan Blade menggunakan argumen kedua pembantu tampilan

Kode program di atas menununjukkan implementasi route yang terdapat pada laravel. Route di atas terdapat pada folder routes dan file bernama web.php. Pada file tersebut, kita dapat membuat route-route baru sesuai keinginan supaya link URL dapat diakses melalui browser.

Templating blade {!! !!} memiliki arti yaitu php echo apabila diterjemahkan ke dalam bahasa pemrograman vanila php biasa.

Penerapan seleksi kondisi Switch Case

Perintah Switch digunakan untuk memeriksa kebenaran suatu nilai dengan memiliki banyak pemilihan atau nilai pengecekannya. Switch case ini akan berjalan saat nilai variable dipanggil atau bernilai true. Apabila ada syntax “break” maka kode blok program akan berhenti. Pada contoh diatas, terdapat sebuah variabel $name yang dikondisikan sama dengan 'budi'. Pada parameter Switch diatas kita mengisikan variabel $name, yang berarti nilai variabel yang ingin diperiksa adalah variabel$name. Jika variable $name sama dengan 'budi' maka ditampilkan “namanya Budi”.

Output Switch Case

$name sebeulmnya didefinisikan dengan isi nilai yaotu 'budi'. Oleh karena itu, case 1 akan bernilai true sehingga kode blok program didalamnya dijalankan sehingga mengeluarkan output tulisan "namanya Budi".

Penerapan looping For

Pada templating blade yang terdapat pada laravel juga memungkinkan untuk melakukan looping dalam hal ini yaitu for loop dengan di awali keyword @forloop dan diakhiri dengan @endfor. Start adalah kondisi pada saat awal perulangan. Biasanya kondisi awal ini berisi perintah untuk memberikan nilai kepada variabel $i. Variabel $i sendiri adalah sebuah variabel yang menentukan berapa banyak perulangan dilakukan. Kebanyakan programmer menggunakan variabel i sebagai variabel counter (ini tidak harus, boleh juga memakai variabel lain). Condition adalah kondisi yang harus dipenuhi agar perulangan berjalan. Selama kondisi ini terpenuhi, maka blade akan terus melakukan perulangan. Misalnya condition ini berisi perintah $i < 10, maka selama variabel $i berisi angka yang kurang dari 10, terus lakukan perulangan.Increment adalah bagian yang dipakai untuk memproses variabel counter agar bisa memenuhi kondisi akhir perulangan. Bagian ini akan selalu di eksekusi di setiap perulangan. Disebut increment karena biasanya berisi operasi increment seperti $i++, yang sebenarnya sama dengan $i = $i + 1. Maksudnya, dalam setiap perulangan naikkan variabel $i sebanyak 1 angka.

Output looping For

DI atas merupakan output dari program sebelumnya dengan menggunakan pengulangan forloop.

Penerapan looping Foreach

Di atas merupakan kode program pada route yang terdapat pada file web.php. Pada kode program tersebut kami membuat route dengan return view dari file yang bernama coba. Namun berbeda dengan sebelumnya, pada file tersebut, membuat array baru yang berisi data mahasiswa meliputi nim dan nama yang dikirimkan kepada file views yang bernama file coba.

Di atas merupakan kode program yang terdapat pada file coba.blade.php yang terdapat pada folder views gunja implementasi dari program looping data menggunakan for each. Setelah sebelumnya data mahasiswa dilirimkan melalui file route, maka file coba akan menerima data tersebut dengan langsung menggunakan looping dengan jenis for each untuk menampilkan ke dalam web browser. Syntax dasar dari penggunaan foreach pada templating blade diawali dengan @foreach lalu menggunakan nama v ariable ayang telah dikirimkan melalui route yaitu $mahasiswa yang diasumsikan dengan keyword as menjadi $mhs. Kemudian di dalam kode blok for each memanggil array nim dan nama dari data mhs dengan cara array. Hal ini dikarenakan pada file route kami mengirimkan datanya dengan menggunakan array, bukan dengan objek,JSON, atau yang lainnya.

Kemanan Form pada Blade PHP

Setiap kali Anda mendefinisikan formulir HTML di aplikasi Anda, Anda harus menyertakan bidang token CSRF tersembunyi di formulir sehingga middleware perlindungan CSRF dapat memvalidasi permintaan. Anda dapat menggunakan @csrf Blade directive untuk menghasilkan bidang token
