Laravel Blade Template

Definisi Blade

Blade adalah fitur yang disediakan Laravel untuk proses templating sederhana namun sangat bermanfaat dalam proses pengembangan tampilan halaman web. Tidak seperti fitur templating PHP populer lainnya, Blade tidak membatasi programmer untuk menggunakan kode PHP biasa dalam membuat kode untuk tampilan. File template blade menggunakan ekstensi file .blade.php dan biasanya disimpan di direktori resources/views.


Routing Blade

Tampilan blade dapat dikembalikan dari rute atau pengontrol menggunakan bantuan tampilan global. Tentu saja, seperti yang disebutkan dalam dokumentasi tentang tampilan, data dapat diteruskan ke tampilan Blade menggunakan argumen kedua pembantu tampilan

Kode program di atas menununjukkan implementasi route yang terdapat pada laravel. Route di atas terdapat pada folder routes dan file bernama web.php. Pada file tersebut, kita dapat membuat route-route baru sesuai keinginan supaya link URL dapat diakses melalui browser.

Menampilkan data yang tidak terhapus pada Blade

Kode program di atas merupakan impelementasi dalam penggunaan templating blade pada framework laravel yaitu menampilkan data yang telah dikirimkan melalui controller maupun route. Templating blade {!! !!} memiliki arti yaitu php echo apabila diterjemahkan ke dalam bahasa pemrograman vanila php biasa.

Output data yang tidak terhapus pada Blade

Apabila file coba.blade.php telah dibuat serta route pada web.php telah ditambahkan yaitu "/coba", maka ketika server lokal diakses kemudian /coba, maka akan menghasilkan output seperti di atas yaitu menampilkan Hello $name. Pada route kita juga harus menambahkan dan mengirimkan data name supaya file views dapat menerima variable tersebut sehingga dapat ditampilkan.

Penerapan seleksi kondisi dengan If Else

Kode program di atas merupakan impelementasi dalam penggunaan templating blade pada framework laravel yaitu penggunaan pengkondisian. Tidak seperti pada source code html biasa, dengan menggunakan templating blade, programmer dapat melakukan beberapa operasi salah satunya pengkondisian if else seperti kode program di atas. Kode program di atas bertujuan untuk mengecek apakah variable $name telah terisi. Jika telah terisi maka kode blok program di dalam statement if akan dijalankan. Penggunaan operasi if pada templating blade menggunakan tanda @if kemudian diakhiri dengan @endif.

Output If Else

Karena variable $name telah terisi yaitu 'budi', maka web browser akan menampilkan data yang terisi dalam variable $name. Dalam hal ini yaitu 'budi'

Rendering Json

Tempalting blade juga memungkinkan programmer untuk menyatukan atau melakukan kolaborasi dengan javascript untuk membuat suatu tampilan supaya blade dapat melakukan rendering terhadap suatu variable yang dikirimkan melalui controller maupun route dengan menggunakan syntax di atas.

Output Rendering Json

Karena sebelumnya $name hanya diisi 'budi'dan bukan berisi syntax kode program javasript, maka web browser tidak menampilkan pesan dan program dapat berjalan tanpa menampiljan pesan eror yang menandakan laravel dapat melakukan kolaborasi dengan javasript untuk melakukan rendering suatu variable yang dikirimkan.

Inheritance views

Kode program diatas merupakan templating blade dalam melakukan pembagian section suatu tampilan views yang terdapat pada laravel. Diawali dengan @extends dengan tujuan untuk memanggil main atau halaman utama tampilan dalam hal ini file bernama container. Kemudian dilakukan inisialisasi @section dengan nama section yaitu coba yang dimana di dalam section coba berisi tag html yaitu h1. Lalu diakhiri dengan @endsection sebagai penutup dari section yang telah dibuat.

Kode program diatas merupakan kode program yang terdapat pada file container.blade.php yang dimana di dalamnya terdapat syntax lengkap dari sebuah html. Namun pada tag body dilakukan inisialisasi bagian dengan menggunakan syntax @yield dengan nama coba. Sub bagian tersebut akan mengambil bagian atau apapun isi dari file coba yang telah dibuat sebelumnya.

Output Inheritance views
